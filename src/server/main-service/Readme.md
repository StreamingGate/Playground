# Main-Serivce
일반 영상, 라이브에서 공통으로 사용되는 기능, 친구 기능을 모아둔 API입니다.

# 사용 기술
* Spring boot 2.6
* MariaDB

# API
* 일반 영상과 라이브 카테고리별 조회 (무한 스크롤)
* 특정 유저에게 저장된 알림 리스트 조회
* 친구 리스트 조회
* 친구 요청, 삭제
* 친구 요청 대기 리스트 조회
* 친구 요청 수락, 거절
* 일반 영상 또는 라이브에 반응하기, 취소하기 (반응: 좋아요, 싫어요, 신고하기)
